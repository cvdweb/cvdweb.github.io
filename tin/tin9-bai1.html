Bài 1: Phương trình quy về phương trình bậc nhất một ẩn

Tóm tắt dạng toán của Bài 1:

1. Phương trình tích

Muốn giải phương trình (a1x + b1)(a2x + b2) = 0, ta giải hai phương trình a1x + b1 = 0 và a2x + b2 = 0, rồi lấy tất cả các nghiệm của chúng.

Ví dụ 1. Giải các phương trình:
a) 5x(x – 11) = 0;

Ta có 5x(x – 11) = 0

5x = 0 hoặc x – 11 = 0

x = 0 hoặc x = 11.

Vậy phương trình đã cho có hai nghiệm là x = 0 và x = 11.

b) (x + 6)(3x – 1) = 0;

Ta có x + 6 = 0 hoặc 3x – 1 = 0

x = –6 hoặc 3x = 1

x = –6 hoặc x = 1/3.

Vậy phương trình đã cho có hai nghiệm là x = –6 và x = 1/3.

Chú ý:

Trong nhiều trường hợp, để giải một phương trình, ta biến đổi để đưa phương trình đó về phương trình tích.

Ví dụ 2. Giải các phương trình:
a) x² – 2x = 0;

Ta có x² – 2x = 0

x(x – 2) = 0

x = 0 hoặc x – 2 = 0

x = 0 hoặc x = 2.

Vậy phương trình đã cho có hai nghiệm là x = 0 và x = 2.

b) (2x + 1)² – 9x² = 0;

Ta có (2x + 1)² – (3x)² = 0

(2x + 1 + 3x)(2x + 1 – 3x) = 0

(5x + 1)(–x + 1) = 0

5x + 1 = 0 hoặc –x + 1 = 0

5x = –1 hoặc –x = –1

x = –1/5 hoặc x = 1.

Vậy phương trình đã cho có hai nghiệm là x = –1/5 và x = 1.

2. Phương trình chứa ẩn ở mẫu quy về phương trình bậc nhất

2.1. Điều kiện xác định của phương trình

Đối với phương trình chứa ẩn ở mẫu, điều kiện của ẩn để tất cả các mẫu thức trong phương trình đều khác 0 gọi là điều kiện xác định của phương trình.

Ví dụ 3. Tìm điều kiện xác định của mỗi phương trình sau:
a) 3/x + 1/(x – 4) = 2;

Điều kiện xác định của phương trình là x ≠ 4.

b) 2/(x + 2) – 3/x + 2 = 1/(2x + 1);

Ta có x + 2 ≠ 0 khi x ≠ –2 và 2x + 1 ≠ 0 khi x ≠ –1/2.

Vậy điều kiện xác định của phương trình là x ≠ –2 và x ≠ –1/2.

Nhận xét:

Những giá trị của ẩn không thoả mãn điều kiện xác định không thể là nghiệm của phương trình.

2.2. Giải phương trình chứa ẩn ở mẫu

Các bước giải phương trình chứa ẩn ở mẫu:

  • Bước 1: Tìm điều kiện xác định của phương trình.
  • Bước 2: Quy đồng mẫu thức hai vế của phương trình, rồi khử mẫu.
  • Bước 3: Giải phương trình vừa nhận được.
  • Bước 4: Xét mỗi giá trị tìm được ở Bước 3, giá trị nào thoả mãn điều kiện xác định thì đó là nghiệm của phương trình đã cho.
Ví dụ 4. Giải các phương trình:
a) 2/(x + 1) + 1/(x – 2) = x – 5/x – 2;

Điều kiện xác định: x ≠ 0 và x ≠ 2.

Ta có:

2x(x – 2)/(x(x – 2)) – (x + 1)(x – 2)/(x(x – 2)) = x(x – 5)/(x(x – 2))

Giải phương trình, ta tìm được x = –1 (thoả mãn điều kiện xác định).

b) 3x – 2 + 2x + 1 = 2x + 5(x – 2)(x + 1);

Điều kiện xác định: x ≠ 2 và x ≠ –1.

Giải phương trình, ta tìm được x = 2 (thoả mãn điều kiện xác định).
